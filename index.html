How it all began ?

In April 2001 a programmer Bram Cohen from University of Buffalo proposed a peer-to-peer protcol, later on 2 July 2001 the first version of BitTorrent was released. The peer-to-peer protocol was named BitTorrents, as well as the file sharing program to use the protocol, known as BitTorrent.

Why Bram Cohen created BitTorrent?

In early life Bram Cohen used to work at MojoNation. MojoNation allowed people to break up confidential files into encrypted chunks and distrubte those pieces on different computers running the software. So if somebody wanted to download a file he have to download it from many sources. Cohen found this concenpt very useful for file shareing and protocol.
Cohen designed BiTorrent to be to download files from many different sources reducing the download time, especially for users with faster download and upload speed. The more popular a file is, the faster a user will be able to download it.
